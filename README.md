### 脱发概况
脱发（**Alopecia**，**Hair loss**），即毛发脱失。多数患者会因美容和心理原因来原因，其也可能是系统性疾病原因。临床中常见的脱发症状有：雄激素性脱发、斑秃、生长期脱发、遗传性毛干发育不良、头癣等。例如斑秃(**Alopecia areata**,**AA**)[[1]](https://doi.org/10.5826/dpc.1304a255)一种常见、非瘢痕性、自体免疫性疾病，人群中占比约[[2]](https://doi.org/10.1016/j.jaad.2017.04.1141.)。并且在斑秃患者中有20%人群都存在家族遗传性症状[[3]](https://doi.org/10.5826/dpc.1304a255)。且该疾病在近期的GWAS研究中表明，与免疫进程相关INF、HLA、T细胞激活与增殖相关基因成关联性[[4]](https://doi.org/10.1159/000530432)。近期Zhou 等人的综述中系统性的描述了斑秃发生的相关免疫调节机制[[5]](https://doi.org/10.1007/s12016-021-08883-0)。

### 间充质干细胞免疫调节与安全性
然而近年来，间充质干细胞（Mesenchymal Stem Cells， MSCs）在治疗自体免疫疾病中初见成效[**免疫调节的综述一篇，后面引两个：糖尿病、银屑病相关文献**]。例如I型糖尿病这一典型的由于自体免疫系统异常而引发的胰岛细胞受损的疾病，其目前已经有大量的实验结果证明经间充质干细胞注射后，机体自身免疫系统得到调控且胰岛细胞得到修复[**引一篇**]。相关I/II期临床试验也证明了其安全性[[6]](https://doi.org/10.1186/s13287-022-02941-w)。在另一自体免疫异常且多基因易感的疾病-银屑病，在相关临床治疗研究中也颇有成效[**引一篇**]。间充质干细胞近年来也被应用在各类免疫性疾病的治疗中，其主要可以有效的通过不同的细胞因子分泌动态调节机体内源性免疫细胞（如NK、DC、巨噬细胞等）、适应性免疫细胞(B细胞、T细胞等)(见图2)[[7]](https://doi.org/10.3390/ijms231710023)。然而近年来部分研究团队认为间充质干细胞自身分泌的外泌体（MSC-exomes，MSC-EXOs)是动态调节免疫系统中免疫细胞行使相关功能的主要作用的关键作用结构[[8]](https://doi.org/10.3390/cells8121605)。

### 毛囊
毛囊是胚胎早期发育过程中原型内胚层-中胚层紧密合作而生成皮肤附加产物和动态小器官。毛囊很容易接近多种来源干细胞例如：表皮干细胞、黑色素细胞干细胞以及间充质干细胞等[[10]](https://doi.org/10.1021/acsbiomaterials.7b00072)。在毛囊发育过程中，其依靠于三个主要阶段：生长期（anagen）、生长中期（catagen，退行期）、静止期（telogen），期间各类干细胞发挥其自身能力调控三个时期阶段的生长发育[[11]](https://doi.org/10.1152/physrev.2001.81.1.449)。期间，生长期2～6年、退行期3个月、休止期2～3个月。

### 脱发的病理生理学

在毛发生长周期的三个阶段，退行期期间大约有50～100根头发脱落。新的毛发开始生长之时，毛囊发育周期又重新启动。
生长周期紊乱导致的脱发：

- 生长期脱发——生长期中断导致毛发异常脱落。
- 休止期脱发——进入休止期头发增多。

#### 分类

可根据有无瘢痕进行分类：
瘢痕是指：毛囊在收到主动性损伤的结果。毛囊受到了不可逆的伤害后被纤维组织取代。

- 瘢痕性脱发：有些在脱发初期表现为双相型，即在发育初期并未有瘢痕性的特点，但随着疾病发生进展进一步形成瘢痕永久性脱发的结果。瘢痕性脱发又可进一步表现为初级形式（即炎症目标为毛囊的卵泡）与二级形式（非特异性炎症导致的卵泡受损）
- 无瘢痕性脱发：即毛发生长速度减少/减慢而形成的脱发（毛囊并未受不可逆损伤）。主要影响毛干的疾病也被认为是无瘢痕性脱发。

### 已知间充质外泌体数据库储备

外泌体[[13]](10.1146/annurev-biochem-013118-111902)即细胞外分泌囊泡，最初在1960s,Bonuci[[14]]()和Anderson[[15]]()发现了软骨细胞所分泌的～100nm左右的囊泡后，渐渐的被人熟知。其主要特点为1）单膜结构；2）直径在30～200nm，但形态学不稳定，尤其是在同一细胞系下也同样有不通结构的外泌体；3）外泌体内部包含糖类、酯类、蛋白、DNA、RNA多种成分；

### 毛囊间充质干细胞分离与制备
#### 参考方案
本参考方案参考了2020年的一篇综述中所提及到的方案[[16]](https://www.ncbi.nlm.nih.gov/pmc/articles/PMC7360986/pdf/WJSC-12-462.pdf)，该论文中主要讲述的三个办法来进行毛囊间充质干细胞的分离与培养。

***hHF-MSCs***的分离最主要需要分离出**完整的毛囊**。

一般情况下分离毛囊细胞需要取头皮组织，然后对头皮组织区使用I型胶原酶进行解离，进而得到完整的毛囊结构。

一个完整的毛囊[[16]](https://www.ncbi.nlm.nih.gov/pmc/articles/PMC7360986/pdf/WJSC-12-462.pdf)包含以下几个结构：1）内根鞘（inner root sheath）2）外根鞘（Outter root sheath）3）结缔组织鞘/真皮鞘（Connective Tissue sheath/dermal sheath）

<img width="578" alt="image" src="https://github.com/bryanwong0610/MSC_exomes_project/assets/53446971/4afaeb33-e077-4ab4-b8d3-a10d1509037d">

一般情况下，头皮所取样本为整体的组织样本：1）上皮；2）真皮；3）真皮脂肪层。毛囊一般定位在脂肪组织区。

**毛囊分离术1**

- 将取下的皮肤组织，使用PBS（1% 青霉素/链霉素）进行冲洗；
- 修建底部脂肪层，将上方组织样本切割成2～4mm小块；
- 使用I型胶原酶（1mg/mL）对组织块进行裂解，37 °C，间隔性搅拌/振动；
- 4h 酶解后，将表皮从组织细胞上剥下，暴露的真皮层中释放出毛囊组织；
- 使用 40mm 细胞过滤器过滤毛囊细胞，PBS冲洗透彻以确保真皮/表皮细胞无残留

**毛囊分离术2**

- 直接从毛囊中拔下毛囊头枕区；
- 使用PBS（1%青霉素/链霉素）进行冲洗；

**hfMSC分离方案1**

- 使用分离术2所分离的毛囊，将每个毛囊去除毛干后，放置于96孔板中；
- 每个孔中加入 100ul DMEM（10% FBS、10ng/ml bFGF），在37°C、5%CO<sub>2<sub/>下进行培养；
- 经细胞自身迁移后，会形成成纤维样细胞贴壁生长。

<img width="799" alt="image" src="https://github.com/bryanwong0610/MSC_exomes_project/assets/53446971/b7d2f324-6750-4115-a70d-9f7bcc948e24">

**HF-MSC分离方案2**

另一种分离方案是使用真皮乳头细胞/真皮鞘进行MSC的分离，以下是详细方案：

- 分离出的毛囊置于0.1% 胶原酶 I 和 0.25% 中性蛋白酶 II 中 37°C中培养30min；
- 光学显微镜下使用30G针将真皮鞘进行分离；
- 使用手术刀片切下真皮鞘后，使用0.1%胰蛋白酶/0.02M EDTA，37°C中培养30min；DEME（10% FBS、10ng/ml bFGF）终止胰酶反应；
- 收集经胰酶处理后的细胞悬液培养基置于EP管内，200G离心5min，去除上清液；
- EP管内底部沉淀使用DEME（10% FBS、10ng/ml bFGF）重悬，得到HF-MSCs。

**DP-MSCs分离方案**

由于真皮乳头细胞会被毛发基质给吞没，因此可以使用倒置技术对真皮乳头间充质干细胞进行分离。

- 使用剪刀，通过乳头细胞上方的基质将毛囊横切，进而得到末梢球；
- 使用针头，倒置末梢球进而暴露出毛发基质和真皮乳头细胞；
- 轻轻去除末梢球上相关的基质和乳头细胞上任何的真皮组织；
- 将清理干净的末梢球置于DMEM（20%FBS），37°C、5%CO<sub>2<sub/>下进行培养；
- 观察贴壁成功，转移至DMEM（10% FBS、10ng/ml bFGF）体系下进行后续扩增培养。

**HF-MSC的扩增**

通常情况下，HF-MSC需要置于**100mm** 培养皿内培养。培养条件：**10ml DMEM（10% FBS、10ng/ml bFGF）**；**37°C、5%CO<sub>2<sub/>**培养箱。
一般情况下，需每间隔2D进行一次换液。
在皿内密度达到90%以后，需使用0.1%胰酶/0.02M EDTA进行消化传代。
